{ เกี่ยวกับฉัน }

สวัสดีครับ ผมชื่อว่า อภิสิทธิ์ ลิขิตวัฒนไพศาล หรือ เรียกว่า เฮียง ก็ได้ครับ ปัจจุบันว่างงาน

ส่วนตัวเป็นคนชื่นชอบการเขียนโปรแกรมมิ่งต่างๆ ที่ไม่ใช่ในเชิงคำนวณจ๋า แต่จะเป็นจำพวก Website หรือจะเป็น Web Application ตามยุคสมัย ชอบท้าทายอะไรใหม่ๆ หรือเรียนรู้อะไรใหม่ๆเสมอมาตั้งแต่เด็ก มีการเรียนรู้และพัฒนาด้วยตัวเองค่อนข้างดี

ปัจจุบันหลังจบการศึกษามัธยมต้น จึงได้เรียนรู้ด้วยตัวเองเกี่ยวกับสิ่งที่ตัวเองชอบเรื่อยมา และพยายามคว้าโอกาสที่พัฒนาตัวเองได้เสมอเรื่อยๆ

ปัจจุบันกำลังศึกษา หรือใช้งานภาษาหรือเทคโนโลยีใหม่ๆ ดังนี้
